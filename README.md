# Bank-Marketing-Dataset-Classification
This repository contains information and material for the Bank Marketing Dataset Classification, a project for the course "Data Analytics" (UNIPI, Department of Informatics). The purpose of the project is to find out which clients will subscribe a term deposit. The project was written in Python. Main tasks where duplicate values detection, missing values detection, statistical analysis for numerical but especially for categorical features (including Density-Histogram-Scatter Plots, QQ-Plots, Distribution Fitting Tests with Kolmogorov-Smirnov, Shapiro-Wilks, Anderson Darling, Correlation Analysis with Pearson Spearman, Kendall), outlier detection (with Z-Score, IQR-Score, QD-Score), anomaly detection (with Isolation Forest), feature engineering, data splitting, data scaling, minority oversampling techniques, clustering (with DBSCAN, K-Means, K-Prototypes and evaluation with davies-bouldin, calinski-harabasz and silhouette), feature selection (with PCA, Mutual Information, X2 and feature importance of the Random Forest) and classification (with Multi-Layer Perceptron, Naive-Bayes, Decision Tree, NEO-LSSVM, KNN, and neural network from keras and evaluation with metrics like Accuracy, F1, Precision, Recall, Roc-Auc).

# The dataset can be found in: https://archive.ics.uci.edu/dataset/222/bank+marketing
